Weather Data Collector

The Weather Data Collector is a serverless application built on AWS that automatically fetches, processes, and stores weather data. The application utilizes AWS Lambda for computation, retrieves data from a public weather API, stores raw data in Amazon S3, and saves processed data in an Amazon RDS PostgreSQL database.
